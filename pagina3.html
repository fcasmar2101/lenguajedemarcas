Instalacion e informacion.

Para instalar Dolibarr en local necesitas un sistema operativo como Windows 10/11, Linux o macOS, y un entorno con Apache/Nginx, PHP (versión 7.4 o superior) y MySQL/MariaDB. También puedes usar paquetes todo-en-uno como DoliWamp (para Windows) o instalarlo manualmente con XAMPP, WAMP o MAMP. El proceso básico de instalación consiste en descargar Dolibarr desde su sitio oficial o clonar el repositorio desde GitHub, descomprimirlo en el directorio raíz del servidor web (como htdocs o www), iniciar el servidor web y acceder a http://localhost/dolibarr para lanzar el instalador. Este asistente guía la configuración de la base de datos, la creación del usuario administrador y los ajustes básicos de idioma, empresa y módulos iniciales. Una vez instalado, puedes acceder al backoffice para activar módulos como Facturación, CRM, Inventario, Compras, Proyectos o Recursos Humanos, desde la sección de configuración. Dolibarr permite gestionar clientes, productos, presupuestos, facturas, almacenes y tareas, todo desde una interfaz web intuitiva. Además, puedes configurar monedas, impuestos, zonas geográficas, y definir roles de usuarios. En cuanto a la seguridad, Dolibarr permite definir perfiles de permisos granulares por módulo, crear grupos de usuarios, y usar HTTPS, autenticación por IP o mediante LDAP u OpenID Connect. También es posible registrar auditorías de acceso y acciones realizadas. Dolibarr incluye generación de informes en formatos PDF, Excel y CSV, exportación/importación de datos desde cualquier módulo, y personalización de documentos mediante plantillas en ODT o PDF con campos dinámicos. También permite ampliar sus funcionalidades con módulos externos desde el DoliStore o desarrollados a medida.
